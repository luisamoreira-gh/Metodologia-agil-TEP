# Metodologia-agil-TEP

Atividade colaborativa com o intuito de explorar a linguagem de programação X, fazendo uso de ferramentas de metodologia ágil.

PONTOS A SEREM EXPLORADOS:
O grupo deverá dividir a pesquisa em tarefas menores, por exemplo:
 - História da linguagem
 - Criador(es)
 - Principais características
 - Exemplos de código
 - Áreas de aplicação
 - Vantagens e desvantagens
 - Empresas que utilizam
 - Curiosidades

Cada tarefa deve conter:
 - responsável;
 - prazo;
 - status;
 - fonte de pesquisa;
 - data de conclusão.

ENTREGÁVEIS:
 - Um documento com as divisões das tarefas (Google Docs);
 - Uma apresentação em Power Point;
 - Crie um Readme.md do repositório
